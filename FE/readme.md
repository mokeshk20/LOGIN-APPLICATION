# User Authentication Module

## Overview

This module provides user authentication functionality, including login and registration forms.

## Features

- User login
- User registration
- Error handling for invalid credentials
- Success messages for successful registration

## Technologies Used

- Frontend: HTML, CSS, JavaScript
